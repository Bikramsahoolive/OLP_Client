Creating a Quiz

To create a quiz, follow these steps:

  1. Click the Add Quiz button to create a new quiz section.
  2. Enter the quiz question in the Question input field.
  3. For each quiz, you can add multiple options:
    • Use the Add Option button to add a new option.
    • Enter option text in the input field for each option.
    • Select one of the options as the correct answer by clicking the radio button next to it.
  4. To remove a quiz question, click the Delete Question button next to the question header.

After adding all the necessary quizzes, submit the form to save your quiz.

Creating an Assignment

To create an assignment, follow these steps:

  1. Click the Add Assignment button to create a new assignment section.
  2. Enter the assignment question in the Question input field.
  3. Provide a detailed answer in the Answer textarea input.
  4. To remove an assignment, click the Delete Assignment button next to the assignment header.

After adding all the necessary assignments, submit the form to save your assignments.

Form Validation

Make sure that each question, option, and answer is filled in correctly. Empty fields or missing correct answers will trigger validation errors.

Additional Features

  • You can add or remove quizzes and assignments dynamically.
  • Each quiz can have multiple options, and you must mark one option as the correct answer.
  • Assignments consist of a question and an answer field for descriptive responses.
